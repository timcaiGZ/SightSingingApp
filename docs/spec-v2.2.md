# Spec: SightSingingApp V2.2
**更新日期**: 2026-05-19  
**基于原型**: v0_design_v2  
**上一版本**: V2.1  

---

## Requirements

### R1 · 导航与 Tab 结构
- R1.1 底部 TabView 包含 4 个 Tab，顺序固定：练习 / 乐理 / 测试 / 我的
- R1.2 Tab 图标使用 SF Symbols（music.note.list / book / checkmark.circle / person.circle）
- R1.3 Tab 激活色 #3B82F6，非激活色 #94A3B8
- R1.4 启动默认显示「练习」Tab

### R2 · 练习模块系统
- R2.1 练习 Tab 展示 5 大模块卡片（ModuleCard），每张卡片顶部有 4px 彩色条
- R2.2 每个模块内列出子练习项，每项右侧显示 Progress Ring（圆环进度）
- R2.3 Progress Ring 颜色规则：≥80% 绿色，≥50% 蓝色，>0% 橙色，0% 灰色
- R2.4 点击练习项 → 进入对应练习页（ExerciseContainerView 或专属页面）
- R2.5 模块色彩标识：听力#3B82F6 / 视唱#22C55E / 节奏#F59E0B / 音程#8B5CF6 / 和弦#EC4899
- R2.6 练习子项清单：

  | 模块 | 子项 | 模式 |
  |------|------|------|
  | 听力训练 | 单音辨认 / 音程听辨 / 和弦辨认 / 节奏辨认 / 旋律听写 | 选择题（单音辨认为键盘输入，旋律听写为键盘输入） |
  | 视唱练习 | 单音视唱 / 音程构唱 / 旋律视唱 / 节奏视唱 | 视唱 |
  | 节奏训练 | 四分音符节奏 / 八分音符节奏 / 十六分音符节奏 / 切分节奏 / 三连音 | 选择题 |
  | 音程训练 | 音程比较 / 音程辨认 / 音程构唱 | 选择题（音程构唱为视唱） |
  | 和弦训练 | 三和弦辨认 / 七和弦辨认 / 和弦转位辨认 | 选择题 |

### R3 · 统一练习容器
- R3.1 顶部胶囊式谱式切换，初始值 = 当前全局谱式设置（来自 @AppStorage），切换为临时覆盖不修改全局
- R3.2 进度圆点：● 已完成（半透明蓝）→ ● 当前（实蓝）→ ○ 未完成（灰），显示「X / Y」；Y = `exerciseItem.totalQuestions`，动态取值
- R3.3 谱式展示区：根据当前（临时或全局）谱式偏好渲染对应谱面
- R3.4 题目提示区：谱式切换器下方显示 `Q{n}：{questionText}`，字号 14pt，前景色
- R3.5 操作行（文字按钮，蓝色，无背景）：`新问题` / `分解（可选）` / `重听` 或 `示范`
- R3.6 底部内容插槽（bottomContent）：放置音乐键盘 / 视唱按钮等
- R3.7 分解按钮仅在非视唱模式且 `showDecompose=true` 时显示
- R3.8 练习结束态：X/Y 全部作答后显示完成弹层（sheet 或 overlay），包含：
  - 正确题数 / 总题数 / 本轮正确率
  - 「再来一轮」按钮（重置进度，重新随机出题）
  - 「返回」按钮（返回模块列表）

### R4 · 三种交互模式

#### R4.1 选择题模式（multipleChoice）
- R4.1.1 AudioPromptCard 在听力题顶部展示：大圆圈播放按钮 + 提示文字，不提前显示音符内容
- R4.1.2 旋律听写额外显示 ReferenceNoteCard（标准音 A4/440Hz 按钮）
- R4.1.3 选项列表未选状态为默认样式
- R4.1.4 选中后立即锁定所有选项（disabled），不可更改
- R4.1.5 正确选项高亮绿色背景（success/10），错误选项高亮红色背景（destructive/10）
- R4.1.6 选中答案后，选项列表下方出现「下一题」主按钮（accent色）；点击进入下一题并刷新进度圆点
- R4.1.7 操作行「新问题」在已答题状态下等价于「下一题」（不重置本轮题目计数）

#### R4.2 键盘输入模式（keyboardInput）
> 适用练习：**单音辨认、旋律听写**（其余听力题为选择题，视唱构唱类为视唱模式）

- R4.2.1 谱面实时反馈区：每次按音名键后，音符即时渲染到谱面（六线谱+简谱 或 五线谱）
- R4.2.2 音乐键盘三列布局：

  | 列 | 内容 | 宽度 |
  |----|------|------|
  | 左列 | 升降号（6个）：无 / ♯ / 𝄪重升 / ♮还原 / ♭ / 𝄫重降 | 固定50pt |
  | 中间 | 3×4 网格：休止符𝄽、C D E F G A B（7键）、休止符𝄾、+8va / -8va / 小节线 | 自适应 |
  | 右列 | ⌫退格 / ⇧（预留）/ 完成 | 固定50pt |

- R4.2.3 升降号选中后按音名键自动附加（如 ♭+B → ♭B），附加后升降号自动重置为「无」
- R4.2.4 ⌫退格：删除最后一个已输入音符（非全部清空）
- R4.2.5 键盘背景色 #D1D5DB，琴键为白色卡片（shadow-sm），升降号列为中灰色（#ADB5BD）
- R4.2.6 已输入音符在谱面展示，不使用 chip 列表
- R4.2.7 「完成」按钮在未输入任何音符时为 disabled 状态
- R4.2.8 点击「完成」触发答案校验，将输入序列与 `correctAnswer` 比对：
  - 正确：谱面音符变绿色 + 出现「下一题」主按钮
  - 错误：谱面音符变红色 + 显示正确答案高亮 + 出现「下一题」主按钮

#### R4.3 视唱模式（sightSinging）
- R4.3.1 目标音符：圆形背景（primary/10，直径96pt），圆内显示音名大字（48pt，primary色）
- R4.3.2 音准指示器量程：**±50 音分**
- R4.3.3 游标颜色三态：
  - |cents| ≤ 10：绿色（准确）
  - 10 < |cents| ≤ 20：橙色（轻微偏离）
  - |cents| > 20：红色（偏离过大）
- R4.3.4 反馈文字：
  - 准确：`音准良好!` + 绿色脉动点
  - 偏低：`偏低 N 音分`（橙色文字）
  - 偏高：`偏高 N 音分`（红色文字）
- R4.3.5 准确区间（绿色条）宽度：轨道总宽的 1/5，居中显示
- R4.3.6 演唱按钮（SingButton）：圆形大按钮（80pt），位于 bottomContent 区
  - 未按下：primary 蓝色背景，文字「按住演唱」
  - 按下中：destructive 红色背景 + 外圈 ping 动画，文字「松开结束」
- R4.3.7 按住开始检测，松开停止并计算得分，立即显示评分结果页；评分公式：
  - `pitchScore`（满分 50）= `max(0, 50 - |演唱期间 centsDiff 平均值| × 1.0)`
  - `rhythmScore`（满分 50）= V2.x 阶段为 35~50 随机占位值，V3.0 接入节拍检测
  - `totalScore` = `pitchScore + rhythmScore`
- R4.3.8 评分等级：95-100「完美!」/ 85-94「优秀」/ 70-84「良好」/ 60-69「及格」/ <60「继续加油」
- R4.3.9 评分颜色：≥85 绿色，≥60 橙色，<60 红色
- R4.3.10 评分页：总分大字 + 等级文字 + 音准/节奏分项卡片 + 「下一题」（主）+ 「重新练习」（次）

### R5 · 谱式系统
- R5.1 全局谱式配置：在「我的 → 谱式选择」中单选，默认「六线谱+简谱」，@AppStorage 持久化
- R5.2 练习页顶部胶囊切换为临时覆盖（本地 State），不写回全局；离开练习页后临时值丢弃
- R5.3 六线谱+简谱模式：6 弦线 + 品位标记（含空弦圈）+ 简谱数字 + 唱名（do/re/mi...）
- R5.4 六线谱支持技法标记，显示在音符上方小字：
  - h（锤击）/ p（勾弦）/ /（滑弦）/ b（推弦）/ ~（颤弦）/ x（哑音）
- R5.5 五线谱模式：5 条线 + 高音谱号 𝄞 + 4/4 拍号 + 音符椭圆+符干
- R5.6 键盘输入模式下，已输入音符实时渲染到当前谱式对应谱面

### R6 · 乐理知识库
- R6.1 乐理 Tab 展示 6 大分类，折叠手风琴样式；分类标题有彩色图标背景
- R6.2 点击分类标题展开/折叠，初始仅「基础乐理」展开
- R6.3 展开后显示知识点列表，每项有标题和简介
- R6.4 知识点详情页（通用）：标题 / 所属分类 / 正文说明 / 五线谱 SVG 示例
- R6.5 「七和弦」知识点使用专属详情页（SeventhChordsDetailView）：
  - 12 调选择器（C / G / D / A / E / B / F# / F / Bb / Eb / Ab / Db）
  - 顺阶七和弦表格：按 I-VII 级展示和弦名称 + 类型（maj7 / m7 / 7 / m7b5）
  - 每个和弦附带吉他指法框图（ChordDiagramView）
- R6.6 「调式」分类包含五度圈（CircleOfFifthsView）交互组件：
  - 外圈 12 大调，内圈 12 关系小调，可点击高亮
  - 选中调显示升降号信息（如「2个升号 F#, C#」）
- R6.7 6 大分类总览：

  | 分类 | 图标背景色 | 知识点数 | 知识点 |
  |------|-----------|---------|--------|
  | 基础乐理 | #3B82F6 | 6 | 认识音符 / 音名与唱名 / 全音与半音 / 音符时值 / 节拍与拍号 / 节奏基础 |
  | 识谱知识 | #22C55E | 4 | 五线谱入门 / 简谱入门 / 六线谱识谱 / 谱号与调号 |
  | 音程 | #8B5CF6 | 4 | 音程的概念 / 吉他常用音程 / 音程的性质 / 音程的听辨技巧 |
  | 和弦 | #EC4899 | 5 | 三和弦 / 七和弦 / 和弦转位 / 吉他和弦指法 / 和弦听辨 |
  | 调式 | #14B8A6 | 3 | 大调音阶 / 小调音阶 / 调式关系 |
  | 节奏 | #F59E0B | 3 | 节拍与拍号 / 常用节奏型 / 三连音与多连音 |

### R7 · 测试系统
- R7.1 测试 Tab 顶部显示统计（3格卡片：已完成 / 平均分 / 时长）
- R7.2 测试列表：标题 / 分类 tag / 题数 / 时限 / 最高分（已测试时显示）
- R7.3 进入测试 → 复用 ExerciseContainerView，mode=multipleChoice
- R7.4 测试结束态：全部作答后显示测试成绩页，包含：
  - 总分（百分制）/ 正确题数 / 用时
  - 「查看解析」按钮（逐题回顾，V3.0 实现）
  - 「返回」按钮
- R7.5 最高分更新：测试结束后，若 totalScore > 历史最高分，更新 SwiftData 记录
- R7.6 中途退出处理：点击返回弹确认对话框（「放弃测试？」），确认后不计入测试记录

### R8 · 我的 Tab
- R8.1 用户信息卡（头像 + 昵称 + 练习天数）
- R8.2 学习统计：2×2 Grid（练习次数 / 平均准确率 / 小时 / 连续天数）；全新用户数值显示「—」
- R8.3 学习设置区：谱式选择（单选组）+ 深色模式（Toggle）
- R8.4 通用设置：音频设置 / 每日提醒 / 学习数据（均为跳转入口）
- R8.5 谱式选择：选中项显示蓝色边框 + 蓝色单选圆点，带说明文字

### R9 · 数据持久化
- R9.1 使用 SwiftData 持久化：学习进度 / 练习历史 / 用户设置 / 测试最高分
- R9.2 App 重启后进度保持

### R10 · 音频系统
- R10.1 AudioEngine：使用 AVFoundation 播放 MIDI 合成音
- R10.2 PitchDetector：使用 YIN 算法实时检测音高，延迟 < 100ms
- R10.3 PitchDetector.thumbPosition() 量程参数：scale = 50.0（±50 音分）
- R10.4 麦克风权限：首次使用视唱功能时请求权限
- R10.5 麦克风权限被拒时：SightSingingView 显示权限提示卡片，包含「去设置开启」按钮（跳转系统设置），不进入检测流程

### R11 · 题库系统
- R11.1 所有练习题目从 `QuestionBank` 统一获取，不硬编码
- R11.2 题目数据结构按模式分为两类：
  - **选择题**：`displayContent / displayLabel / question / options: [String] / correctAnswer: String`
  - **键盘输入题**：`displayContent / displayLabel / question / options: [] / correctAnswer: String`（correctAnswer 为目标音名，如 "E4"；options 为空数组）
- R11.3 提供 `QuestionBank.random(for: exerciseId) -> ExerciseQuestion` 接口随机取题；题库为空时返回默认兜底题
- R11.4 题库覆盖所有 exerciseId（见 R2.6 练习子项清单）
- R11.5 听力类选择题的 displayContent 为通用描述（如「听辨单音」），不提前暴露正确答案

### R12 · 和弦框图
- R12.1 `ChordDiagramView`：6 弦 × 4 品网格，Canvas 绘制
- R12.2 支持：空弦（○）/ 不弹（×）/ 按弦位置（实心圆）/ 可选手指编号
- R12.3 用于乐理「七和弦」专属详情页

### R13 · 空状态与错误状态
- R13.1 题目加载失败（QuestionBank 返回 nil）：显示「加载失败」提示 + 「点击重试」按钮
- R13.2 麦克风不可用（权限被拒/硬件错误）：显示权限提示卡片（见 R10.5）
- R13.3 我的 Tab 全新用户：统计数值显示「—」而非 0，引导文字「开始练习后将显示数据」
- R13.4 测试列表为空时：显示「暂无可用测试」占位插图 + 说明文字

## Acceptance Criteria

### AC1 · 导航
- [ ] **4 个** Tab 切换正常，无内容重叠
- [ ] 每个 Tab 内导航栈（push/pop）正常工作
- [ ] 导航栏「返回」按钮可以正确回到上一页

### AC2 · 练习流程
- [ ] 选择题：选模块 → AudioPromptCard 播放 → 选答案 → 绿/红反馈 → 「下一题」出现 → 进入下一题
- [ ] 键盘输入：输入音名 → 谱面实时渲染 → 「完成」→ 正确变绿/错误变红 → 「下一题」出现
- [ ] 10 题完成后弹出完成弹层（正确率 + 「再来一轮」 + 「返回」）
- [ ] 进度圆点随当前题号更新，Y 值与 exerciseItem.totalQuestions 一致
- [ ] 题目来自 QuestionBank，每次随机取题

### AC3 · 谱式切换
- [ ] 进入练习页时胶囊初始值 = 全局谱式设置
- [ ] 胶囊切换后谱面立即更新，题目数据不变
- [ ] 键盘输入模式下切换谱式，已输入音符在新谱式中重新渲染
- [ ] 离开练习页后临时切换不影响全局设置
- [ ] 全局设置在 App 重启后保持

### AC4 · 视唱评分
- [ ] 按住期间指示器实时更新（±50 音分量程）
- [ ] 游标颜色符合三态规则（≤10 绿 / ≤20 橙 / >20 红）
- [ ] 松开后显示评分结果页（总分 + pitchScore + rhythmScore）
- [ ] pitchScore = max(0, 50 - 平均|centsDiff|)，精确到整数
- [ ] 评分等级和颜色符合规则
- [ ] 「重新练习」回到视唱状态，「下一题」进入下一道题

### AC5 · 键盘输入
- [ ] 6 个升降号按钮均可独立激活，附加后自动重置为「无」
- [ ] ⌫退格删除最后一个音符（非全部清空）
- [ ] 未输入时「完成」按钮为 disabled
- [ ] 已输入音符实时渲染到谱面（六线谱+简谱 或 五线谱）
- [ ] 正确 → 谱面变绿 + 「下一题」；错误 → 谱面变红 + 显示正确答案 + 「下一题」

### AC6 · 乐理系统
- [ ] 6 大分类全部展示，总知识点数 = 25（6+4+4+5+3+3）
- [ ] 七和弦知识点进入专属详情页，12 调切换正确更新顺阶七和弦表
- [ ] 每个和弦显示对应吉他指法框图
- [ ] 五度圈可交互，点击调名正确显示升降号信息

### AC7 · 测试系统
- [ ] 测试完成后显示成绩页（总分 / 正确题数 / 用时）
- [ ] 测试最高分在 SwiftData 中更新
- [ ] 中途退出弹确认对话框，取消不中断测试

### AC8 · 持久化
- [ ] 练习进度在 App 重启后保持
- [ ] 谱式偏好在 App 重启后保持
- [ ] 测试最高分在 App 重启后保持

### AC9 · 空状态与错误状态
- [ ] QuestionBank 返回空时显示「加载失败 + 重试」
- [ ] 麦克风权限被拒时显示权限提示卡片，不进入检测
- [ ] 全新用户我的 Tab 统计显示「—」

### AC10 · 性能
- [ ] 页面切换 < 300ms
- [ ] 音高检测延迟 < 100ms
- [ ] 谱面渲染（键盘输入后）< 50ms
- [ ] App 冷启动 < 2s

## Non-functional Requirements
- Platform: iOS 17+
- Framework: SwiftUI + SwiftData
- Deployment: iPhone 竖屏
- 支持 VoiceOver 辅助功能
- 深色模式适配（V2.x 可后期完善）

---

## Out of Scope（V2.x 不做）
- 用户账号体系 / 云端同步
- 社交 / 分享功能
- 付费内购
- Android 版本
- 节拍准确率检测（rhythmScore V3.0 实现）
- 测试逐题解析（V3.0 实现）
- AI 生成题目
