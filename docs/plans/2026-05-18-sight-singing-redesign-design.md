# SightSingingApp 全面重构设计方案 v1.0

> **日期**: 2026-05-18
> **版本**: v1.0
> **目标**: 本周内对标 Solfeggio，实现赶超
> **参考应用**: Solfeggio（练耳大师）App Store ⭐4.6

---

## 一、项目概述

### 1.1 目标定位
- **面向用户**: 民谣吉他进阶学习者
- **核心功能**: 视唱 + 练耳综合训练
- **差异化**: 六线谱 + 简谱双谱式支持（对比 Solfeggio 仅五线谱）

### 1.2 重构目标
本周完成四个维度的全面升级：
1. **界面设计** — 参考 Solfeggio 现代演进风格
2. **视唱功能** — 实时音准检测 + 专业评分
3. **课程模块** — 完整吉他课程学习路径
4. **乐理模块** — 可搜索知识库

---

## 二、设计决策

| 决策项 | 选择 | 原因 |
|--------|------|------|
| 视觉风格 | B. 现代化演进 | 保留 Solfeggio 专业调性 + 现代圆角/阴影/动效 |
| 音准检测 | A. AVFoundation | 免费、苹果原生、快速实现 MVP |
| 重构方式 | A. 重写核心 | 复用 SwiftData 模型和 ViewModel，聚焦 UI + 视唱 |
| 设计主题 | 深蓝专业色调 | 参考 EarMaster / Solfeggio 专业风格 |

## 五、功能模块设计

### 5.1 练习模块（6 大维度）

| 模块 | 吉他场景 | 练习类型 |
|------|---------|---------|
| **音名** | 简谱辨认、六线谱音符识别 | 单音辨认、空弦辨认、根音辨认 |
| **音程** | 把位音程、推弦依据 | 音程辨认、把位比较、击勾弦音程 |
| **和弦** | 常用和弦按法、转换 | 大横按辨认、和弦转换、七和弦 |
| **调式** | 各调音阶把位、CAGED系统 | 调式辨认、音阶练习 |
| **节奏** | 扫弦节奏型、分解和弦 | 常见节奏型、节拍稳定性 |
| **旋律** | 旋律线弹奏、简谱视奏 | 简谱模唱、旋律辨认 |

### 5.2 视唱功能流程

```
简谱展示 → 播放示范 → 用户演唱 → 实时检测 → 评分反馈
                              ↓
                        音准曲线可视化
                              ↓
                        分项得分展示
```

**评分标准**:
```
总分 = 音准分 × 0.7 + 节奏分 × 0.3

音准评分（基于音分偏差）:
- ≤ 10 音分:  100 分
- 10-30 音分: 70-99 分线性递减
- 30-50 音分: 60-69 分
- > 50 音分:  0-59 分
```

### 5.3 课程模块结构

```
课程 (Course)
├── 章节 1 (Chapter)
│   ├── 课时 1.1 (Lesson)
│   │   └── 练习 (Exercise)
│   ├── 课时 1.2
│   └── 课时 1.3
├── 章节 2
│   ├── 课时 2.1
│   └── 课时 2.2
└── 章节 3

预置课程:
- 乐理基础 (3章节/9课时)
- 视唱入门 (3章节/7课时)
- 节奏训练 (3章节/6课时)
```

### 5.4 谱式切换

```
支持三种谱式切换（Tab 切换）:
- 简谱: 1 2 3 4 5 6 7 + 时值标注
- 六线谱: 吉他专用，弦号 + 品位 + 指法
- 五线谱: 可选（对标 Solfeggio）

切换组件:
[简谱] [六线谱] [五线谱]
```
