1. 引言

1.1 目的

本文档定义面向民谣吉他进阶学习者的视唱练耳移动应用的功能需求和非功能需求，作为开发团队与指导老师之间的沟通桥梁，明确项目范围、功能边界和质量要求。

1.2 范围

本项目是一款 iOS 原生应用，名为 SightSingingApp，核心功能包括：

  • 三种谱式（五线谱、六线谱、简谱）的展示与切换
  • 听力训练（单音、音程、和弦、调式、节奏、旋律）
  • 视唱练习与实时音准检测
  • 结构化课程学习路径
  • 学习进度统计与分析

1.3 定义与缩写

术语定义
视唱根据乐谱演唱的技能
练耳通过听觉识别音高、节奏等的技能
六线谱吉他专用谱表，展示弦号和品位
简谱用数字1-7表示音高的记谱法
音分音高最小计量单位，1半音=100音分
音准偏差演唱音高与目标音高的音分数差异

2. 总体描述

2.1 产品定位

一款面向民谣吉他进阶学习者的视唱练耳App

差异化定位：
• 参考 Solfeggio 极简 iOS 设计风格
• 差异化支持六线谱 + 简谱双谱式展示
• 填补市场空白（现有练耳App无吉他谱支持）

2.2 参考应用对比

维度Solfeggio（参考）SightSingingApp（本项目）
目标用户通用音乐学习者民谣吉他进阶学习者
默认谱式仅五线谱六线谱 + 简谱
六线谱支持无真实吉他谱（弦号+品位+指法）
Tab结构自由练习/课程/乐理/我的对齐参考应用
课程结构4层深度对齐，内容适配吉他场景

2.3 参考应用核心设计亮点

  1. 统一练习容器 — 所有题型共用同一页面结构，减少认知负担
  2. 三种交互模式 — 选择题/键盘输入/视唱，各司其职
  3. 课程体系 — 4层深度结构化学习路径
  4. 极简视觉 — 纯白背景、灰色分隔线、iOS HIG规范
  5. 进度圆点 — ●○○○○ 简洁直观

2.4 目标用户

用户画像特征描述
吉他学习者已掌握基础和弦转换，有进阶视唱练耳需求
音乐爱好者希望系统学习音乐听力，提升音乐素养
艺考备考生需要视唱练耳专项训练

2.5 技术约束

  • 平台: iOS 17+
  • 框架: SwiftUI
  • 数据存储: SwiftData
  • 音频处理: AVFoundation
  • 图表: SwiftUI Charts
  • 图标: SF Symbols（线条风格）
  • 主题: 浅色优先，支持深色模式

5. 界面设计规范

5.1 设计理念

iOS HIG原生风格：纯白背景、简洁分区线、列表式布局、底部Tab导航。大量留白，呼吸感强。参考App Store五线谱视唱练耳App的极简专业设计。

5.2 视觉风格要点

  • 整体风格: 纯白背景、简洁列表分类、iOS HIG规范
